NX-KS Mod-pack
如何安装:
      1) 准备一张SD卡，相机开启蓝牙
      2) 解压所有文件到SD卡根目录
      3) SD卡插入相机，并把相机开机
      4) 耐心等待重启1-2次
      5) 当Mod安装完成会提示[已安装完成]
      6) 开机后屏幕上会显示NX-KS图标
		- (如果15秒没没有显示图标，请打开蓝牙)
      x)卸载Modt: 主菜单> 设置> 卸载 Mod

打开Mod菜单方法：点击屏幕上的图标  或者 双击EV（+/-）按键
部分功能有Bug，例如导致卡死，拔出电池即可。
部分功能为二进制文件，需要进行编译源码汉化，因此暂未进行汉化。

禁止利用原版Mod、本汉化Mod进行任何形式的盈利活动。
